Scalable SRAM-Based LUT Architecture

High-Density FPGA Fabric Design

📌 Introduction

FPGA is a reconfigurable hardware device used for digital systems. LUT is the fundamental building block. This project designs a scalable SRAM-based LUT architecture to improve performance and efficiency.

🎯 Objectives

Design scalable LUT
Improve speed
Reduce power
Increase utilization

⚙️ Methodology

SRAM stores logic
MUX selects output
Hierarchical design
Fracturable LUT

📊 Results

  • Delay reduced by ~25–30%
  • Power reduced by ~20–25%
  • Logic utilization increased by ~30%
  • Improved scalability
